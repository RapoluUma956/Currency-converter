
# Currency Converter

A modern, user-friendly web application for converting currencies in real time. This project allows users to quickly convert amounts between different currencies using up-to-date exchange rates fetched from a reliable API. The interface is intuitive, visually appealing, and responsive, making it suitable for both desktop and mobile users.

## Features
- Real-time currency conversion with live exchange rates
- Support for a wide range of global currencies
- Automatic flag display for selected countries
- Clean, modern, and responsive UI
- Error handling for invalid input and network issues

## Technologies Used
- HTML5, CSS3, JavaScript (ES6)
- [ExchangeRate-API](https://www.exchangerate-api.com/) for currency data
- Font Awesome for icons

## How to Use
1. Enter the amount you wish to convert.
2. Select the source and target currencies from the dropdown menus.
3. Click "Get Exchange Rate" to view the latest conversion result.
